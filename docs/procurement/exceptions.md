# Procurement Exceptions

Exceptions are manual, audited requests for known procurement rule violations.

Exception types:

- budget overrun;
- missing price;
- supplier minimum not met;
- supplier maximum exceeded;
- order frequency violation;
- urgent purchase;
- manual override;
- other.

Every exception requires a reason. Pending exceptions do not satisfy gates. Approved exceptions can satisfy the relevant gate condition when policy allows it.

An exception does not approve a proposal, create a supplier order, approve email, send email or select carrier.
