# Pilot Go-Live Checklist

- [ ] Readiness passed.
- [ ] Import dry-run passed.
- [ ] Calculation checked with real SKU.
- [ ] Order proposal checked by supply manager.
- [ ] Manufacturer form export checked.
- [ ] Supplier email draft checked.
- [ ] Email send approval tested in log mode.
- [ ] Inbound confirmation sample processed.
- [ ] Form autofill sample checked.
- [ ] Confirmation application checked.
- [ ] Carrier quote comparison checked.
- [ ] Carrier selection checked by user.
- [ ] Logistics receiving checked.
- [ ] Audit reviewed.
- [ ] Health check reviewed.
- [ ] Backup verified.
- [ ] External integrations approved or disabled.
- [ ] Owner signs off.

Approving the pilot for live does not activate integrations automatically.
