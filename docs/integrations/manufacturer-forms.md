# Manufacturer Forms

## Purpose

Manufacturer form onboarding stores real order templates privately and maps supplier order data into previewable fields.

## Workflow

1. Upload XLSX, XLS, CSV or PDF template.
2. Store the file under private storage.
3. Save header and item mapping.
4. Preview against a supplier order.
5. Export with an available renderer or follow portal manual instructions.

## Renderer Status

Excel rendering is a configured boundary. If PhpSpreadsheet is unavailable, the renderer reports not configured.
PDF rendering is a placeholder.
Portal manual instructions generate human copy/paste steps without browser automation.

## Safety

Real manufacturer files belong in storage, not git.
