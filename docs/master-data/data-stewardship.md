# Data Stewardship

Data steward assignments identify users responsible for reviewing master data issues.

Assignments can target:

- company-wide product master data;
- supplier;
- category;
- product;
- supplier-product mapping.

Stewards review unknown SKUs, pending aliases, duplicate suggestions, merge proposals and change requests.

If notification workflows exist, stewardship can be used as routing context. Steward assignment itself does not approve changes automatically.
