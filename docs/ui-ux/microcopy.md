# UI Microcopy

## Disabled Actions

- Email must be approved before sending.
- Proposal cannot be approved while unresolved review items remain.
- Carrier selection requires user confirmation.
- Apply is disabled until the run is validated.

## Human Review

- Requires human review.
- Review the source evidence before applying this value.
- Resolve all blocking items before continuing.

## AI

- AI suggestions are not final values.
- Accepting extraction does not apply business changes.
- Low confidence requires human review.

## Transport

- System recommendation is not automatic carrier selection.
- User must select carrier.

## Calculation

- Safety stock covers only T2-T3.
- Safety stock covers only T2-T3 and must not duplicate T1-T2.

## Integrations

- Real integrations are disabled by default.
- External AI is disabled unless explicitly approved.
- Secrets are masked and never shown in UI.
